# 3.2. User Stories

| US01              | Acceso a la cuenta                                                    | Como usuario de la aplicación. Quiero acceder a mi cuenta de usuario. Para poder verificar mi identidad Y ver los datos de la persona cuidada.                                                       | **Escenario: Ingreso incorrecto de datos** Dado que el usuario está en la pantalla de acceso. Cuando ingrese la contraseña y/o usuario incorrecto. Entonces la página mostrará un mensaje de "Datos incorrectos". **Escenario: Ingreso correcto de datos** Dado que el usuario está en la pantalla de acceso. Cuando ingrese la contraseña y/o usuario correcto. Entonces el usuario será enviado a la página principal. Y se enviará un mensaje al correo electrónico afiliado.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |             |
| US02              | Registro de cuenta.                                                   | Como usuario de la aplicación. Quiero poder registrarme en la aplicación. Para poder comenzar con el monitoreo de la persona a cuidar.                                                               | **Escenario: Creación de la cuenta** Dado que el usuario está en la pantalla de registro. Cuando el usuario ingrese datos como correo electrónico, nombre de usuario y contraseña Entonces una nueva cuenta se crea Y el usuario es llevado a la página principal. **Escenario: Activación de la pulsera.** Dado que el usuario tiene una cuenta Y hay una pulsera afiliada a la cuenta. Cuando la pulsera se prenda. Entonces la pulsera empezará a actualizar los datos en la base de datos.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |             |
| US03              | Edición de datos de la cuenta.                                        | Como usuario de la aplicación. Quiero poder editar ciertos datos de mi cuenta como contraseña. Para poder corregir errores Y renovar la contraseña.                                                  | **Escenario: Editar nombre de usuario.** Dado que el usuario está en la pantalla de actualización de datos. Cuando el usuario reingrese el nombre Entonces el nombre del usuario será actualizado. **Escenario: Editar contraseña.** Dado que el usuario está en la pantalla de actualización de datos. Cuando el usuario ingrese la contraseña nueva Y reingrese la contraseña anterior Y verifique a través de un correo electrónico. Entonces se actualizará la contraseña de la cuenta. **Escenario: Editar el correo electrónico afiliado.** Dado que el usuario está en la pantalla de actualización de datos. Cuando el usuario reingrese el correo electrónico anterior  Y ingrese la contraseña para verificar Entonces se actualizará el correo electrónico afiliado.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |             |
| US04              | Obtención de la información sobre la persona cuidada.                 | Como usuario de la aplicación. Quiero poder recibir información de la pulsera. Para poder tener constancia de la situación de la persona que estoy cuidando.                                         | **Escenario: Conexión con la pulsera.** Dado que la pulsera está prendida Cuando este reciba una confirmación de afiliación con una cuenta. Entonces la pulsera se afiliará con una cuenta. **Escenario: Guardado de los datos en una base de datos local.** Dado que la pulsera está afiliada a una cuenta Y se encuentra prendida. Cuando se detecte algún cambio en el pulso cardiaco,  presión arterial o distancia de cierto punto. Entonces se actualizarán los datos dentro de la base de datos local.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |             |
| US05              | Lectura de la información sobre la persona cuidada.                   | Como usuario de la aplicación. Quiero poder visualizar la información de la pulsera. Para poder tener constancia de la situación de la persona que estoy cuidando Y poder tomar medidas al respecto. | **Escenario: Acceso a la pantalla de datos** Dado a que el usuario se encuentra en la pantalla principal. Cuando el usuario presione en el botón de "visualización" Entonces el usuario será enviado a una pantalla en dónde de visualiza en tiempo real la presión arterial, el pulso cardiaco y la distancia a cierto punto de la pulsera. **Escenario: Línea de tiempo de presión arterial** Dado a que el usuario se encuentra en la pantalla de visualización de datos. Cuando presione en la sección de presión arterial. Entonces será llevado a una pantalla dónde hay una línea de tiempo que recopila la presión arterial de los dos últimos días. **Escenario: Línea de tiempo de pulso cardiaco** Dado a que el usuario se encuentra en la pantalla de visualización de datos. Cuando presione en la sección de pulso cardiaco. Entonces será llevado a una pantalla dónde hay una línea de tiempo que recopila el pulso cardiaco los dos últimos días.                                                                                                                                                                                                                                                                                                                 |             |
| US06              | Generar alarmas cuando se detecten emergencias en la persona cuidada. | Como usuario de la aplicación. Quiero recibir alarmas en caso que se suceda alguna emergencia. Para poder tomar medidas al respecto.                                                                 | **Escenario: Lectura anormal de pulso cardiaco.** Dado a que la pulsera se encuentra afiliada a una cuenta Y está prendida. Cuando se detecte una pulso cardiaco alto o bajo (mayor a 100bpm o menor a 60 bpm). Entonces la pulsera enviará una notificación con alarma al usuario con detalles del incidente, incluyendo posición, pulso cardiaco y hora del incidente. **Escenario: Lectura de presion arterial alta.** Dado a que la pulsera se encuentra afiliada a una cuenta Y está prendica. Cuando detecte una presión arterial alta o baja (mayor a 140/90 mmHG o menor a 90/60 mmHG) Entonces la pulsera enviará una notificación con alarma al usuario con detalles del incidente, incluyendo posición, presión arterial y hora del incidente. **Escenario: Lectura de distancia límite.** Dado a que la pulsera se encuentra afiliada a una cuenta Y está prendida Y tiene un punto de referencia al cual referir la distancia. Cuando detecte que el punto de referencia está a una distancia mayor (alrededor de 15 metros, pero depende del usuario) Entonces la pulsera enviará una notifiación con alarma al usuario con detalles del incidente, incluyendo posición al momento de generarse la emergencia, presión arterial, pulso cardiaco y hora del incidente. |             |
